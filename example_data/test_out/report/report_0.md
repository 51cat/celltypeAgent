## 3. 模型推理核心观点对比
> 展现不同 AI 模型在逻辑推演上的异同点。

### 🤖 GPT-5.4
* **核心定义**: Naive / Central Memory CD4+ T cells
* **逻辑**: 该模型强调 CCR7、TCF7、LEF1 三者共同反映"淋巴结归巢 + 干样未分化"的 T 细胞状态，结合 PBMC 组织背景，推断最可能为 Naive 或中央记忆样 CD4+ T 细胞；同时注意到 COL18A1 和 ITGA6 存在混合信号，但认为不足以推翻主体注释，体现了较为审慎的推理风格。

### 🎨 Claude-Sonnet-4-6
* **核心定义**: Naive CD4+ T cell
* **逻辑**: 该模型着重突出 MAL 基因的亚型特异性，认为 MAL 是 Naive CD4+ T 细胞的高特异性标志，与 LEF1、TCF7 组合形成强证据链，明确倾向 Naive 而非中央记忆状态；同时指出缺少 CD4 蛋白本身的直接表达证据，存在不能完全排除 Naive CD8+ T 细胞的轻微不确定性，推理逻辑最为细致严谨。

### 🔬 Qwen3.5-397B-A17B
* **核心定义**: Naive T cells
* **逻辑**: 该模型聚焦于 CCR7 与 TCF7 的协同高表达作为 Naive T 细胞的核心判据，推理简洁直接；但未进一步区分 CD4/CD8 亚类，对 MAL 等 CD4 偏向性 Marker 的讨论较少，亚型分辨率相对保守。

### 🚀 Grok-4.2
* **核心定义**: CD4+ 幼稚 T 细胞
* **逻辑**: 该模型对 MAL 基因的 CD4+ 特异性给予了较高权重，与 CCR7、LEF1、TCF7 一同构成"CD4+幼稚"状态的证据四元组；推理逻辑与 Claude 接近，明确指向 CD4+ 亚型，且特别强调了 PBMC 人类图谱背景的先验支持。

### 🧠 GLM-5
* **核心定义**: Naive T cells
* **逻辑**: 该模型在注释结论上与 Qwen 相近，未明确区分 CD4/CD8，但补充提及了 ITGA6 和 COL18A1 在 Naive T 细胞中的背景表达，对 COL18A1 的非特异性表达做出了合理解释，体现出对潜在干扰信号的关注。

### 💡 DeepSeek-V3.2
* **核心定义**: Naive / Central Memory T cells
* **逻辑**: 与 GPT-5.4 相似，该模型对 Naive 与中央记忆状态持并列态度，认为 CCR7 单独无法区分两者；同时额外提及 COL18A1（胶原）和 ITGA6（整合素）可能涉及迁移或细胞互作，为 Marker 基因列表中的"非典型"基因提供了一定的功能解释，思路相对全面。

## 4. 审核意见与最终结论 (Audit Consensus)
> 结合顶级专家审核意见，对组织背景和模型偏倚进行最终校正。

> [!IMPORTANT]
> **最终鉴定结论：Naive CD4+ T cells（静息未分化功能状态）**
> 全部 6 个模型在 T 细胞主谱系层面完全一致，共识比例达 1.0，信息熵为 0.0，可靠性评分均为 88 分，最终确定该 Cluster 为 **Naive CD4+ T cells**。

### 核心判定依据：

1. **基因保真度校核（Gene Faithfulness）**：所有 6 个模型的推理均通过 Gene Faithfulness 审核，未发生任何 Gene Hallucination，全部证据基因（CCR7、LEF1、TCF7、TRAT1、MAL、IL6ST、TRABD2A、RCAN3 等）均源自原始 Marker Gene List，推理可信度基础扎实。

2. **核心 Marker 组合的亚型判定**：`CCR7 + TCF7 + LEF1` 三基因共表达是 Naive/中央记忆 T 细胞的经典特征；进一步引入 `MAL` 基因（文献证实在 Naive CD4+ T 细胞中特异性高表达）后，证据链明确偏向 **CD4+ Naive** 而非 CD8+ 或效应型亚群，支持最终亚型判定为 Naive CD4+ T cells。

3. **组织背景合理性**：该 Cluster 来源于外周血单个核细胞（PBMC），Naive CD4+ T 细胞是 PBMC 中丰度最高的淋巴细胞亚群之一，与已发表的人类 PBMC 单细胞转录组图谱完全吻合，生物学背景高度合理。

4. **不确定性与后续建议**：
   - ⚠️ Marker 基因列表中 **`COL18A1`** 和 **`ITGA6`** 在 T 细胞中表达相对非特异，各模型审核均指出此两基因引入了轻微混合信号，建议后续验证时对这两个基因进行独立表达分布检查，排除潜在 doublet 污染。
   - ⚠️ 由于缺乏 **`CD4`** 蛋白编码基因的直接高表达证据，CD4 亚型判断主要依赖 MAL 等间接 Marker，建议在条件允许时结合 CITE-seq 蛋白质层面数据或流式验证加以确认。
   - ✅ 整体而言，该 Cluster 注释质量高，结论可信，可直接用于下游差异表达分析与细胞通讯推断。
